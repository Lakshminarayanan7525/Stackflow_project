# Stackflow Project

A Full Stack Web Application developed using React.js for frontend and Spring Boot for backend.

## 🚀 Features
- Product Management System
- Add Products
- View Products
- Frontend and Backend Integration
- REST API Support

## 🛠 Technologies Used

### Frontend
- React.js
- JavaScript
- HTML
- CSS

### Backend
- Spring Boot
- Java
- Maven

## 📁 Project Structure

StackFlow/
│
├── product/frontend      # React Frontend
├── productinventory      # Spring Boot Backend

## ▶️ How to Run

### Frontend

```bash
cd product/frontend
npm install
npm start
```

### Backend

```bash
cd productinventory/productinventory
mvn spring-boot:run
```
